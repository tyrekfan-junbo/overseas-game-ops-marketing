# Report Style

## Voice

Write for a game operations lead. Be concise, direct, and decision-oriented.

Avoid:

- Process notes.
- “This update added...” paragraphs.
- Debug/tool details.
- Repeated strategy paragraphs across pages.
- Unsupported performance claims.

Prefer:

- Finding, evidence, implication, action.
- Tables for schedules, tests, sample classifications, and platform roles.
- Charts for sample mix, channel distribution, priority mix, and stage coverage.
- Screenshots or video frames when visual evidence affects the decision.

## Recommended Report Structure

1. Executive conclusion.
2. Product and stage judgment.
3. Competitor/material evidence.
4. Official social operations.
5. UA and creative test plan.
6. Store/channel launch plan.
7. Data tracking and KPI requirements.
8. 30/60/90-day or stage-based action plan.
9. Source appendix.

## Writing Rules

- Make the current stage explicit.
- Separate evidence from inference.
- Keep competitor observations tied to Yeeyo actions.
- Preserve local source paths in appendix or source tables.
- When data is insufficient, state the smallest next collection step.
- If two sections repeat, merge them and keep the more actionable version.
